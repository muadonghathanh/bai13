BÀI 13 – CSS

Tổng hợp lý thuyết – luyện tập – vận dụng

Luyện tập 1

Câu hỏi: Khẳng định “Ngôn ngữ định dạng CSS chính là ngôn ngữ HTML” là đúng hay sai?

Lời giải:

Sai. CSS không phải HTML. HTML tạo cấu trúc nội dung, còn CSS điều chỉnh giao diện (màu sắc, font, bố cục...). CSS là ngôn ngữ độc lập hoạt động cùng HTML.

Luyện tập 2

Câu hỏi: “Chỉ cần thay đổi tệp CSS là có thể thay đổi định dạng nhiều trang web” đúng hay sai?

Lời giải:

Đúng. Một file CSS có thể áp dụng cho nhiều trang web. Khi chỉnh file CSS → toàn bộ giao diện trang dùng nó sẽ thay đổi.

Vận dụng 1

Câu hỏi: Nêu sự giống nhau giữa Style Sheet trong phần mềm soạn văn bản và CSS của trang web.

Lời giải:

  • Cấu trúc – cú pháp: Đều tuân theo quy tắc định dạng rõ ràng.
  • Tính tái sử dụng: Dùng cho nhiều đoạn / nhiều phần tử.
  • Khuôn mẫu định dạng: Tạo style cho tiêu đề, đoạn văn, danh sách…
  • Kiểm soát phong cách: Điều chỉnh màu, cỡ chữ, khoảng cách…

Vận dụng 2 – Lịch sử CSS

Lịch sử CSS

Ý tưởng CSS do Hakon Wium Lie (Na Uy) đề xuất năm 1994 khi làm việc với Tim Berners-Lee tại CERN.

Mục tiêu của CSS là tạo ra một ngôn ngữ định dạng riêng, tách biệt hoàn toàn khỏi cấu trúc HTML.

Lịch sử các phiên bản CSS đầu tiên

• CSS1 ra đời năm 1996.

• CSS2 hoàn thành năm 1998, bản cập nhật hoàn thiện nhất là CSS2.1 (2011 — nâng cấp 2016).

Các phiên bản CSS tiếp theo

Từ CSS3 trở đi, ngôn ngữ được phát triển theo từng module riêng. Một số chuẩn CSS4, CSS5 hiện vẫn đang phát triển.

Chuẩn web (HTML, CSS…) do W3C phát triển: www.w3.org
